# Бак с водой

Интерфейс представляет собой прямоугольник, под которым находится 2 кнопки,
наполнить контейнер и опустошить. При нажатии кнопки наполнить, через каждые 2 секунды, контейнер наполняется на 20% до тех пор, пока не наполнится полностью.
Аналогично, при нажатии кнопки опустошить, каждые 2 секунды, вода в баке уменьшается на 20%. 
Необходимо отображать все 4 возможные статуса (опустошен, наполняется, наполнен и опустошается)
Нажатие опустошить или наполнить при статусах наполняется или опустошается соответственно, должно изменять статусы и производить указанные действия.

```
    // start
    yarn
    yarn dev
```
